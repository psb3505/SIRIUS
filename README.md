<h3>culiso는 Bluetooth로 가전제품(BLE 키트)에 연결하여, AI 챗봇이나 수동 방식으로 제어하고, 사용자 간 정보와 노하우를 커뮤니티 게시판을 통해 공유할 수 있는 플랫폼입니다.</h3>
<h3>챗봇 대화 기록 및 커뮤니티 관련 기능은 Spring Boot 기반으로 개발된 RESTful 백엔드 서버를 통해 처리되도록 구현하였으며, 이 서버는 Docker Container 위에서 배포될 수 있도록 구성했습니다.</h3>
